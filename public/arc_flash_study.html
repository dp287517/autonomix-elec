Analyse d'Arc Flash

Comment utiliser cette page

  • Utilisez les filtres pour sélectionner un bâtiment ou un tableau spécifique.
  • Consultez le tableau ci-dessous pour voir les résultats de l’analyse d’arc flash pour chaque disjoncteur.
  • Chaque ligne affiche l’énergie incidente, la distance de protection, et le niveau de PPE requis.
  • Pour les données manquantes, cliquez sur la loupe pour identifier les disjoncteurs concernés.
  • Cliquez sur l’icône ou "Agrandir" pour voir un graphique détaillé.
  • Le schéma interactif montre les relations entre tableaux et disjoncteurs. Déplacez les nœuds ou double-cliquez pour éditer.
  • Les lignes rouges signalent un risque élevé (énergie incidente > 8 cal/cm²).
  • Cliquez sur "Exporter en CSV" pour télécharger les données.
