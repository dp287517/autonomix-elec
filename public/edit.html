Modifier un tableau électrique

Étapes pour modifier un tableau

  1. Vérifiez l'identifiant du tableau (modifiable, doit être unique).
  2. Indiquez si le tableau est le général principal du site.
  3. Modifiez les disjoncteurs existants via le bouton "Modifier" (enregistré immédiatement).
  4. Ajoutez des disjoncteurs en :
    • Recherchant avec OpenAI (marque et référence).
    • Sélectionnant un disjoncteur existant (marque et référence).
    • Entrant les données manuellement.
  5. Transférer un disjoncteur vers un autre tableau via le bouton "Transférer".
  6. Lier un disjoncteur non principal à un tableau aval via le champ "Tableau aval".
  7. Vérifiez les conflits d'ID (en rouge).
  8. Désignez un disjoncteur comme principal.
  9. Enregistrez les modifications pour les nouveaux disjoncteurs ou l'ID du tableau.
