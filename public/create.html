Créer un nouveau tableau électrique

Étapes pour créer un tableau

  1. Entrez l'identifiant du tableau (unique, ex. 40-8-G pour BT, HTA-01 pour HTA).
  2. Indiquez si le tableau est HTA ou BT, et remplissez les champs HTA si applicable.
  3. Indiquez si le tableau est le général principal du site (enregistré automatiquement).
  4. Ajoutez des équipements (disjoncteurs, transformateurs, cellules MT, câbles) en :
    • Recherchant via OpenAI (marque et référence pour disjoncteurs).
    • Sélectionnant un équipement existant (marque et référence).
    • Entrant les données manuellement.
  5. Complétez les champs, en suivant les instructions (ex. ID obligatoire, courbe = B, C ou D pour disjoncteurs).
  6. Désignez un disjoncteur comme principal si nécessaire.
  7. Lier un disjoncteur non principal à un tableau aval si applicable (BT uniquement).
  8. Enregistrez le tableau en cliquant sur "Enregistrer".
