Créer un nouveau tableau électrique

Étapes pour créer un tableau

  1. Entrez l'identifiant du tableau (unique, ex. 27-9-G).
  2. Indiquez si le tableau est le général principal du site (enregistré automatiquement).
  3. Ajoutez des disjoncteurs en :
    • Recherchant via OpenAI (marque et référence).
    • Sélectionnant un disjoncteur existant (marque et référence).
    • Entrant les données manuellement.
  4. Complétez les champs, en suivant les instructions (ex. ID obligatoire, courbe = B, C ou D).
  5. Désignez un disjoncteur comme principal si nécessaire.
  6. Lier un disjoncteur non principal à un tableau aval si applicable.
  7. Enregistrez le tableau en cliquant sur "Enregistrer".
