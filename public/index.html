Bienvenue sur Autonomix Elec

Cet outil vous permet de gérer vos tableaux électriques facilement.

Comment utiliser l'outil

  • Consultez cette page pour comprendre l'outil.
  • Allez à la page Créer un tableau pour ajouter un nouveau tableau électrique.
  • Entrez les informations du tableau, comme son identifiant (ex. 27-9-G) et les disjoncteurs associés.
  • Utilisez l'API OpenAI pour récupérer automatiquement les caractéristiques des disjoncteurs en entrant la marque et la référence.
  • Enregistrez le tableau dans la base de données.
  • Consultez la page Voir les tableaux pour modifier ou supprimer un tableau existant.
  • Visitez la page Sélectivité pour analyser la sélectivité entre les disjoncteurs et tableaux.
  • Consultez la page Obsolescence pour suivre la durée de vie des disjoncteurs et planifier les remplacements.
  • Accédez à la page Évaluation du Niveau de Défaut pour analyser les courants de court-circuit et compléter les données nécessaires.
  • Explorez la page Évaluation des Risques pour identifier les risques électriques (arcs, chocs, surchauffes) et mettre à jour les données manquantes.
  • Consultez la page Organigramme Maintenance pour visualiser la structure du département de maintenance.
  • Utilisez la page Urgence Distribution pour signaler et gérer les pannes électriques critiques.
  • Générez des rapports détaillés via la page Rapports pour exporter vos analyses en PDF.
  • Accédez à la page Programme de Sécurité Électrique pour gérer les procédures de sécurité, la maintenance préventive et les formations du personnel.
